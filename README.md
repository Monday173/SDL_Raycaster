# SDL Raycaster 
***A basic raycaster made using pure C and SDL.***

## Custom Maps
***This raycaster requires a map file to be passed as a command line argument.***

**Map Format**
```
<player_x> <player_y>

<num_colours>

<r> <g> <b>
...

<map_width> <map_height>

<tile> <tile> <tile> ...
<tile> <tile> <tile> ...
...
```
